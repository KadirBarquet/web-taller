¿Qué son HTML CSS y JS?

Son los tres lenguajes que pueden dominar el desarrollo de un sitio web,
o al menos su parte visual e interactiva.

HTML:

Es un lenguaje de marcado que puede describir lo que debe mostrarse en un sitio web
utilizando etiquetas y atributos.

CSS:

Es la forma de describir el aspecto y diseño de una página web mediante selecciones de
etiquetas y sintaxis para los diseños como por ejemplo los colores para los textos.

JavaScript:

Es un lenguaje de programación que puede permitir a los programadores crear páginas web
Este puede utilizarse para poder crear animaciones, juegos y todo tipo de aplicaciones web.
